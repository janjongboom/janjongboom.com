Jan Jongboom

Hi! I'm a software developer / evangelist currently working on Firefox OS and Internet of Things applicances for Telenor Digital. Within the Firefox OS project I focus on my work as a peer for the keyboard and on performance, especially on low-cost devices like the 25$ smartphone. Writing code for both Gaia (the UI layer) and Gecko (underlying engine). For a grasp of my day-to-day work, see my bugzilla profile. I am also part of Project Gonzo, which is a camera running on my fork of Firefox OS: JanOS.

Besides being doing hardcore coding I do evangelism on Firefox OS, HTML5 development and offline web applications. Both in conferences (Yandex YAC, JSConf EU, JSConf Asia, VelocityConf, etc. etc.), as well as training events; app days; competitions; hackathons on Firefox OS in Telenor markets.

I've also worked at Cloud9 IDE, funda.nl and at a bunch of smaller companies. My CV has it all. Occasionally I also write a blog post (personal, Gonzo), and help people on StackOverflow. My code (the open source part at least) is in patch files on bugzilla or on GitHub. You can contact me on twitter, or through janjongboom@gmail.com.
